Politique de confidentialité

Dernière mise à jour : Février 2026

1. Collecte de données

L'application Préparation TCF Canada collecte un minimum de données pour fonctionner correctement :

  • Pseudo : Votre prénom ou pseudo, stocké localement.
  • Progression et Objectifs : Vos scores, votre Objectif Quotidien (Daily Goal), et vos statistiques de l'outil Time Capsule.
  • Historique de Tests : Vos résultats des Tests Complets et entraînements.
  • Enregistrements Audio : Vos réponses orales (sauvegardées uniquement sur votre appareil).

Toutes les données sont stockées localement. Aucune donnée n'est transmise d'office vers un serveur externe.

2. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Personnaliser votre entraînement avec le suivi de l'Objectif Quotidien.
  • Analyser vos statistiques avec l'outil Time Capsule.
  • Vous permettre de réécouter vos performances orales.
  • Générer des Tests Complets adaptés à votre historique.

3. Services tiers

L'application utilise les services suivants :

  • Apple App Store : Pour la distribution et les achats intégrés.
  • Synthèse vocale iOS : Pour la lecture audio des exercices de compréhension orale.

Aucun service d'analyse tiers ou de publicité externe n'est intégré dans l'application.

4. Stockage et sécurité

Toutes vos données sont stockées localement sur votre appareil iOS et protégées par les mécanismes de sécurité d'Apple. Aucune donnée n'est stockée sur des serveurs distants.

La suppression de l'application entraîne la suppression définitive de toutes les données associées.

5. Droits de l'utilisateur

Vous avez le droit de :

  • Modifier votre pseudo à tout moment dans les paramètres.
  • Supprimer toutes vos données en désinstallant l'application de votre appareil.
  • Nous contacter pour toute question relative à vos données.
